## Run Locally

Clone the project

```bash
  git clone https://github.com/jaskarnvir/Typing-Pro.git
```

Go to the project directory

```bash
  cd Typing-Pro
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the app

```bash
  Use Pycharm to run the project
```
